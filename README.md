# phonebook

## Project setup:

### Front end:

```
cd phonebook
```

```
npm install
```

### Back end:

```
cd API
```

```
npm install
```

#### Compiles and hot-reloads for development

##### Front end:

```
npm run serve
```

##### Back end:

```
npm start
```
